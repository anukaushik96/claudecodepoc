# Calculator Web App

A sleek, modern calculator built with HTML, CSS, and JavaScript. Features a dark theme with smooth animations and keyboard support.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Percentage calculations
- Sign toggle (+/-)
- Decimal point support
- Keyboard input support
- Responsive design
- Modern dark UI with subtle animations

## How to Use

1. Open `calculator/index.html` in your web browser
2. Use the on-screen buttons or your keyboard to perform calculations
3. Press `=` or `Enter` to calculate the result
4. Press `AC` or `Escape` to clear all
5. Use `Backspace` to delete the last digit

## Technologies

- HTML5
- CSS3 (with modern features like CSS Grid and Flexbox)
- Vanilla JavaScript (no frameworks required)
